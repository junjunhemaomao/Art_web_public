# 《冥府诡契：红丝缚魂》角色形象设计

## 一、追猎者阵营（中式恐怖化身）

### 1. 红煞·阴婚新娘（控场型）
**形象特征**
- 清末江南嫁衣造型：正红色绸缎嫁衣(RGB:190,30,45)绣有金线凤凰纹样，裙摆浸透暗红色血丝(RGB:100,10,15)与黄褐色尸油(RGB:180,160,50)，形成不规则污渍
  - 写实细节：嫁衣金线采用真实刺绣工艺表现，每根丝线有0.1mm粗细差异；血渍边缘呈现干涸龟裂效果，尸油污渍有深浅不一的渗透层次
- 面部特征：血红色盖头(RGB:150,0,0)下隐约可见腐烂面容 - 左脸完整保留新娘妆容，右脸肌肉组织脱落露出森森白骨
  - 写实细节：左脸妆容粉底有细微毛孔和汗珠，右脸白骨表面有微生物腐蚀形成的蜂窝状结构
- 手部细节：十指延伸出5米可伸缩红绸丝线(RGB:180,40,60)，丝线表面有类似血管的凸起纹路，末端尖锐如针
  - 写实细节：丝线血管纹路随张力变化而膨胀收缩，针尖有0.5mm倒刺结构
- 材质质感：
  - 嫁衣绸缎：强反光材质(金属度0.3 粗糙度0.2)，光线直射时呈现丝绸光泽，织物经纬线密度为300根/英寸
  - 血丝：半透明凝胶材质(透明度0.7)，内部有缓慢流动的粒子效果，每毫升含约5000个红细胞模拟物
  - 尸油：油性材质(金属度0.8 粗糙度0.1)，在光照下呈现彩虹色折射，粘度系数为0.85Pa·s
- 动作特征：
  - 移动：漂浮时嫁衣以0.5秒间隔无规律飘动，丝线末端有0.2秒延迟跟随
  - 攻击：右手前伸→丝线快速延伸(0.3秒)→缠绕目标(0.5秒)→收缩(0.4秒)
- 环境互动：
  - 雾气：丝线经过处留下1秒残留的红色雾痕
  - 光线：强光下尸油产生彩虹色光晕，血丝会吸收周围红光

### 4. 傩尸匠·赶尸人（召唤型）
**形象特征**
- 湘西赶尸人装束(RGB:80,60,40)，背后竹篓装满僵尸(粗糙度0.7)

- 面部贴满褪色符咒(RGB:180,80,80)，右眼为空洞(深度0.9)
  - 写实细节：符咒每张尺寸9×9cm±0.3cm误差，朱砂褪色程度约60-70%，眼窝空洞边缘有0.1-0.3mm骨裂痕迹
- 手持摄魂铃(金属度0.8)与招魂幡(亚麻材质 粗糙度0.6)
  - 写实细节：铃铛内壁有0.2mm厚铜绿沉积(铜绿生长速度0.01mm/年)，幡布每厘米经纬线密度为18×22(误差±1)
- 材质质感：
  - 竹篓：编织纹理(法线贴图强度0.5)，表面有霉斑效果(霉斑直径1-3mm，密度20-30个/cm²)
  - 符咒：半透明羊皮纸材质(透明度0.3±0.05)，边缘有烧焦痕迹(焦痕宽度2-5mm，碳化深度0.1-0.5mm)
  - 铃铛：青铜材质(金属度0.8 粗糙度0.3)，内部有幽绿磷光(RGB:0,180,100±5)，磷光强度随铃声变化±15%(频率2-3Hz)
- 动作特征：
  - 移动：每走三步摇铃一次(0.5±0.1秒间隔)，幡布以0.3±0.05秒延迟飘动(摆动幅度15-30度)
  - 攻击：摇铃(0.4±0.05秒)→僵尸群起(0.6±0.1秒)→包围目标(0.8±0.1秒)
- 环境互动：
  - 雾气：铃声响过处产生1±0.2秒绿色雾圈扩散(直径1.5-2m)
  - 月光：招魂幡会吸收月光产生冷光效果(亮度0.3-0.5lux)

  - 写实细节：胭脂盒描金线条宽度0.3-0.5mm(误差±0.05mm)，迷香粒子密度约2000±100颗粒/立方厘米
- 脖颈处有未愈合的缝合线(法线贴图强度0.6±0.05)
  - 写实细节：缝合线每厘米8-10针(误差±0.5针)，针脚处有0.1-0.3mm组织液渗出
- 材质质感：
  - 皮肤：蜡质材质(金属度0.1 粗糙度0.4±0.02)，表面有细微血管纹理(直径0.05-0.1mm)，毛孔密度约150±10个/平方厘米

- 动作特征：
  - 移动：每3±0.2秒变换一次容貌(0.8±0.05秒渐变过程)，步伐轻盈无痕(步幅0.5-0.8m)
  - 攻击：打开胭脂盒(0.3±0.02秒)→释放迷香云雾(0.6±0.05秒)→目标眩晕(1±0.1秒)
- 环境互动：
  - 光线：强光(>1000lux)下皮肤会呈现半透明效果(透明度增加20-30%)
  - 阴影：在暗处(<10lux)缝合线会发出微弱红光(RGB:200±5,0,0，亮度0.2-0.3cd/m²)
