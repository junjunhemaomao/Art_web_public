# 🌆《神遗：蒸汽妖都》主角小队职业设计

> ✨ 世界观关键词：娲术：上古女娲创世生物基因技术、蒸汽时代、电气发明、血族、山海经神兽
> ✨「娲术」作为​​反派专属的禁忌技术​​不被主角滥用，同时强化主角团以​​蒸汽科技、电气科技、武道气术、神兽共鸣​​等合法手段对抗娲术污染的对抗逻辑。​
      ​主角的力量来自对正统技艺的磨砺或对自身悲剧的克制，而非驾驭禁忌
> 🎮 战术配置：经典7人队伍。同时最多4人出战

---

## 🛡️【前线位】稳定战线 / 扛伤打断

### 1. 陈少秋（Chen Shaoqiu）  
“一人之身，承古今之道；一念之间，破乱世之局。”   
武术世家子弟`帝国学院归国少年陈少秋，在回国后的不久，被卷入娲术灾变中。既熟练传统气术，也掌握现代战斗战术，是队伍中的领导者与攻守枢纽。  
他的人生观坚守“护人先护道”，因此他在战斗中也承担平衡队形、分析局势、干预敌态的多功能角色。

- **职业类型**：天衡之刃（平衡型剑士）  
- **定位**：近战/指挥/多功能型，兼具输出与辅助，核心为战场节奏控制  
- **风格灵感**：苍炎之轨迹的艾克 + 克劳德（FF7）+ 八方旅人的剑士
- **武器**：中国剑，灵枢冲击者,特斯拉电枪／压缩霰弹
- **普通攻击**：
  - **衡剑斩**：单体物理攻击，冲锋斩击目标并使其瘫痪，有中等几率破防（降低目标护甲1回合） 
- **技能**：
  - **脉冲护体**：开启电场护盾，短时间内减伤 + 嘲讽敌人  
  - **云势·崩**：单体高威力物理技，额外削减敌方行动条   
  - **破曦·斩**：觉醒技，满气力释放，对敌方全体造成大范围斩击

## ✨【支援位】治疗恢复 / 状态净化 / 增益增伤

### 5. 林晚晴（Lin Wanqing）  
- **职业类型**：气络医者（Qi-Flow Healer）  
- **定位**：群体治疗 / 异常驱散 / 附加增益  
- **风格灵感**：塞拉（FF13）+ 神圣法师 + 和风巫医
- **武器**：银针阵 / 火药炸弹 / 气脉传导环  
- **普通攻击**：
  - **毒针**：单体物理攻击，并带3回合毒性伤害 
  - **净血注射**：治疗量最低队友恢复生命
- **技能**：
  - **百草清瘴**：驱散单体异常状态  
  - **银针渡穴**：快速为全队恢复生命  
  - **龙息灸法**：3回合内持续为全体恢复生命，并提升抗性 
  - **金针定脉**（极技）：禁锢敌人气流，使其瘫痪数回合

## ⚔️【输出位】远程压制 / 爆发暗杀

### 3. 阿伊莎（Aisha）  
情报与潜入专家  
- **职业类型**：隐毒刃客（Veilblade Assassin）  
- **定位**：高机动杀手 /毒状态叠加、暴击率高  
- **风格灵感**：影牢女王 + 潜行型的P5小丑
- **武器**：炼毒短刃/ 飞镖
- **普通攻击**：
  - **刃鸣**：单体物理攻击，并带3回合毒性伤害 
- **技能**：
  - **幻影潜行**：后背攻击，高暴击率  
  - **蛇骨镖**：中毒投掷物，附加麻痹效果  
  - **毒谶契印**：多段攻击次敌人，若目标中毒，则暴击率翻倍

  - **风华突**：高命中率单体穿刺，击中后降低目标闪避率1回合 
- **技能**：
  - **特斯拉电涌**：发射电网，造成群体电击  
  - **蒸汽阱缚**：布置地面陷阱，2回合禁锢目标，并持续2回合伤害  
  - **智识**：为队友下回合攻击叠加伤害

### 6. 维克多（Victor）⭐新增小正太角色  
14岁音乐神童，用音乐声波的神秘力量
- **职业类型**：谐律操纵者（Resonance Conductor）
- **定位**：全队增益 / 节奏控制 / 音波攻击  
- **武器**：音叉权杖 + 八音盒（背包装置）  
- **普通攻击**：
  - **律动打击**：单体音波伤害，20%概率使目标行动条倒退10% 
- **技能**：
  - **震魂调**：发出高频谐波，使受娲术控制的敌人有概率陷入混乱，无差别攻击同类 
  - **加速谐波**：全队行动条推进15%，下次伤害+20% 
  - **圣咏屏障**：3回合音波护盾，吸收伤害的30%转化为治疗 
  - **​​终末共鸣（极技）**：对异常状态目标进行7段音爆连击

### 7. 芙蕾雅（Freya）⭐新增兽耳小萝莉角色  
被植入「娲灵」神兽基因的沉默实验体。外表是天真无邪的萝莉，内心封印着神兽的低语。   
能与古神兽产生短暂共鸣。她是战场的异数：不依赖娲术，却因其副产物而变强。
- **职业类型**：谐律操纵者（Resonance Conductor）
- **定位**：高爆发AOE / 神兽召唤 / 心智抗性 / 异常反击  
- **风格灵感**：九尾、玉藻前 + 神隐萝莉 + 神兽共鸣器
- **武器**：神兽羽晶 / 音律浮翎 / 灵脉断指甲  
- **普通攻击**：
  - **律灵爪轻击**：用爪状灵力劈击单体，造成法术伤害，有小概率施加1回合的「扰神」效果（使敌人技能冷却+1回合） 
- **技能**：
  - **神语共鸣**：与远古神兽短暂同步，为全队施加“灵兽庇佑”（2回合内全属性抗性+20%，敌人命中-10%） 
  - **九尾炎煌**：召唤九尾神兽幻影，对敌方全体造成火属性AOE伤害，并灼烧3回合（每回合固定扣血） 
  - **祸斗破魂**：召唤祸斗灵影冲锋敌方单体，造成高威力精神伤害，若敌方正处异常状态，则附加1回合“惧意”（无法使用主动技能）
  - **​禁忌回响（极技）**：神兽基因短暂完全激活，变身3回合，提升自身50%攻击、免疫所有控制效果；技能替换为“灵爆连击”，每回合自动释放高爆发AOE攻击一次（不耗技能点），并自动驱散1个自身不良状态。
