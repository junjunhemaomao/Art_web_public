# 物品设计  
30+ Items

## 命名规则
**格式**：`[分类前缀]_[描述名称]`  
- 分类前缀：根据物品类型分组，明确分类。  
- 描述名称：用简洁的英文描述物品内容，采用首字母大写的驼峰命名法（CamelCase）。  

---

## 分类与物品列表

### 1. 武器类（Weapons）  
**前缀**：`Weapon_`  
**说明**：角色战斗使用的主要装备。  
**示例**：  
| 名称 | 描述 |
| ---- | ---- |
| Weapon_Sword | 基础剑，攻击力适中，初始武器 |
| Weapon_Bow | 弓箭，可远程攻击 |
| Weapon_FlameSword | 火焰剑，附带火焰伤害效果 |
| Weapon_IceAxe | 冰霜斧，攻击时减速敌人 |
| Weapon_LightningDagger | 闪电匕首，攻击附带连锁闪电效果 |

---

### 2. 防具类（Armors）  
**前缀**：`Armor_`  
**说明**：提升防御力或特殊抗性的装备。  
**示例**：  
| 名称 | 描述 |
| ---- | ---- |
| Armor_LeatherVest | 皮甲，提供基础防御力 |
| Armor_SteelArmor | 钢甲，提升物理防御力 |
| Armor_MagicRobe | 魔法长袍，增强法术抗性 |
| Armor_FlameShield | 火焰盾，减少火焰伤害 |
| Armor_IceHelmet | 冰霜头盔，免疫冰冻状态 |

---

### 3. 消耗品类（Consumables）  
**前缀**：`Item_`  
**说明**：一次性使用的物品，恢复或增强效果。  
**示例**：  
| 名称 | 描述 |
| ---- | ---- |
| Item_HealthPotion | 恢复少量生命值的药水 |
| Item_ManaPotion | 恢复魔法值的药水 |
| Item_Antidote | 解毒药，解除中毒状态 |
| Item_SpeedBoost | 临时提升移动速度 |
| Item_PowerElixir | 短时间内提升攻击力 |

---

### 4. 材料类（Materials）  
**前缀**：`Material_`  
**说明**：用于制作或交易的原材料。  
**示例**：  
| 名称 | 描述 |
| ---- | ---- |
| Material_Wood | 普通木材，基础建造材料 |
| Material_IronOre | 铁矿石，可锻造装备 |
| Material_MagicCrystal | 魔法晶石，提升附魔成功率 |
| Material_FlameCore | 火焰核心，制作火属性装备 |
| Material_Herb | 草药，炼制药水的原料 |

---

### 5. 工具类（Tools）  
**前缀**：`Tool_`  
**说明**：可反复使用的道具，帮助探索或战斗。  
**示例**：  
| 名称 | 描述 |
| ---- | ---- |
| Tool_Torch | 火把，照亮黑暗区域 |
| Tool_Bomb | 炸弹，用于破坏障碍物 |
| Tool_Hookshot | 钩爪，用于到达高处或远处 |
| Tool_Map | 地图，显示当前区域详情 |
| Tool_Compass | 指南针，指引目标方向 |

---

### 6. 特殊物品类（Special Items）  
**前缀**：`Special_`  
**说明**：与任务或解谜相关的独特物品。  
**示例**：  
| 名称 | 描述 |
| ---- | ---- |
| Special_AncientKey | 古代钥匙，开启隐藏的房间 |
| Special_MagicStone | 魔法石，解锁传送门 |
| Special_TreasureMap | 藏宝图，指引隐藏宝藏的位置 |
| Special_RoyalSeal | 王室印章，与主线任务相关 |
| Special_SacredArtifact | 神圣遗物，解锁最终挑战 |

| 武器类        | `Weapon_`     | `Weapon_FlameSword`   |

通过该分类和命名规则，可以快速找到特定物品类型并组织管理物品资源。 

---

### 分类说明
1. **武器**：用于战斗，如剑、弓、魔杖等。
2. **防具**：提高防御力，减少伤害，如盾牌、护甲等。
3. **消耗品**：一次性使用物品，提供即时效果，如药水、卷轴等。
4. **工具**：用于探索或解谜，如钥匙、地图、火把等。
5. **材料**：掉落或收集的道具，用于制作或强化装备。
6. **特殊道具**：推动剧情或提供独特功能的关键物品。

通过这些物品设计，游戏玩法可以涵盖战斗、探索、解谜和成长，丰富玩家体验。
