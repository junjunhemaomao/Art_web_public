
# 青云镇 
演武堂：位于小镇中心，是镇长居住和处理事务的地方，也是传授武功的主要场所。大堂气势恢宏，雕梁画栋，堂内供奉着武学先辈的牌位。

悟真阁：位于演武场东侧，是一座三层的木质楼阁，楼身被翠绿的藤蔓环绕，充满了古朴的韵味。阁中珍藏着瑞云村的武学秘籍以及从各处收集来的武学心得。阁内有专门的守阁长老，守护着这些珍贵的典籍。

神兵阁：对应 “武器店”，位于村子中心区域附近。店内陈列着各式各样的兵器，从常见的刀剑枪棍，到奇门兵刃，应有尽有。店老板是位资深的铸剑师，不仅售卖兵器，还能根据顾客需求进行定制和修缮，兵器的锋利程度与工艺在江湖上都颇有名气。

奇珍阁：对应 “道具店”，摆满了琳琅满目的物品，有提升内力的丹药、增加防御力的软甲、隐匿身形的夜行衣，以及各种江湖中实用的小道具。掌柜的消息灵通，能淘到不少江湖罕见的奇珍异宝，供侠士们选购。

醉仙楼：对应 “酒馆”，村中的热闹之地，木质的建筑，酒旗飘扬。店内提供各种美酒佳酿，还伴有特色的下酒菜。江湖中人常在此相聚，谈天说地，交流江湖轶事、切磋武功心得，偶尔也会有冲突摩擦在此上演。

凌云剑派：对应 “有国王的城堡”，原城主府位置。门派建筑气势恢宏，楼阁错落有致，位于村子地势较高处，易守难攻。凌云剑派以剑术闻名，门规森严，庇护着瑞云村免受匪盗侵扰。门派内设有练武场、藏经阁、议事厅等。练武场供弟子们日常修炼剑术，藏经阁收藏着门派的剑谱、内功心法以及各类武学典籍，议事厅则是门派高层商议要事、接待来访宾客的地方。掌门德高望重，带领着门下弟子守护一方，同时也会接纳村中资质上佳的年轻人入门修炼。

悦来客栈：对应 “旅店”，为往来的江湖人士提供住宿和饮食。客栈房间干净整洁，分为普通客房和上房，满足不同需求。店小二热情好客，能为客人提供不少村子周边的信息。
