# S_NE组件拼合规则矩阵
基础组件：S_NE（小方块，带北墙和东墙）

## 北边(N)拼合可能性
| 可拼合组件 | 结果描述 | 空间效果 |
|------------|----------|----------|
| S_N, S_NE, S_NW | 墙体延续 | 形成连续的北向墙面 |
| S_S, S_SE, S_SW | 对向墙 | 形成南北向通道或房间 |
| S_F | 延展地板 | 向北扩展内部空间 |
| S_E, S_W | 丁字墙 | 形成T形墙结构 |
| S_NS, S_NEW, S_NSW, S_NESW | 复杂墙体 | 形成复杂空间分隔 |

| S_E, S_NE, S_SE | 对向墙 | 形成东西向通道或房间 |

## 组合效果说明
1. **墙体延续**: 形成连续的外墙或内墙
2. **对向墙**: 创建房间边界或通道
3. **延展地板**: 扩大内部空间面积
4. **丁字墙**: 形成空间分隔或转角
5. **复杂墙体**: 创建更复杂的空间结构

## 特殊考虑
- 所有拼合都需考虑3D模型的中心点对齐
- 确保最终形成封闭的室内空间
- 考虑游戏玩法需求的空间尺寸
- 注意不同组件组合可能产生的视觉效果

## 使用建议
1. 优先考虑基本的墙体延续和对向墙拼合
2. 使用地板延展来创建更大的内部空间
3. 合理使用丁字墙和复杂墙体来丰富空间变化
4. 注意维持空间的游戏性和可通行性
