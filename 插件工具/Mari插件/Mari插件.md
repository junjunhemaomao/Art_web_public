# Mari插件

- [MariUsdPlugins](https://github.com/TheFoundryVisionmongers/MariUsdPlugins?utm_source=chatgpt.com)    
支持 Mari 与 USD 的导入/导出、USD 场景图处理     
Foundry 官方维护，随着 Mari 的 USD 支持一起演进   
- [JorelLatraille / J-Tools](https://github.com/JorelLatraille/J-Tools?utm_source=chatgpt.com)    
一系列用于 Mari 的 Python 脚本工具    
- [MariMe](https://github.com/cgbeige/MariMe)    
Maya ↔ Mari 间传输几何 / 材质 / 通道，支持 UDIM 等     
作者写得较早，可能需要适配最新版 Mari / Maya 环境     
- [HoloSuite / HoloEdit Mari Tools](https://learn.arcturus.studio/docs/holoedit/en/v2021.2/html/External/Mari_tools.html?utm_source=chatgpt.com)    
包含 Import / Export / Painting Scripts，支持按帧 / 分层处理体数据     
文档中提及兼容 Mari 4.6v2 / 4.7 等旧版本     
- [Mari Extension Pack (MARI Extension Pack / mariextensionpack)](https://campi3d.com/External/MariExtensionPack/userGuide5R8/RequirementsInstallation.html?utm_source=chatgpt.com)    
扩展 Mari 的功能，比如 Smart Mask 工具、Shading 辅助工具等    
属于扩展包，可能需要根据你用的 Mari 版本选用相应版本
