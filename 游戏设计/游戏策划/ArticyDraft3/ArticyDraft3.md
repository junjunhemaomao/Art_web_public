# Articy:Draft 3  
![alt text](./image.png)
> articy:draft 3 可以用于叙事与任务设计，其流程结构支持各种复杂多变的游戏叙述风格。它并不仅仅能够用于设计结构，而且可以直接作为创作工具。当我们将之比拟为 Scrivener 的时候，articy:draft 3 的好处也是明显：尤其对于非线性写作而言，类似工具虽然并不帮助我们写作，但在厘清思路这一方面要比 Excel 强出太多。

我认为这个玩意的优势在于，可以脱离于项目进行编辑。
也就是说策划可以在实现功能前，自行组织逻辑，至少保证在配置层面可以跑通，而不是直接跑去跟程序说要什么功能。

除此之外，articy:draft 3 还有其他优点：
支持 Unity 与 Unreal 的内容导出
高度自定制导出
宏插件系统
脚本物体属性支持
无限层级
.JSON 导出
可扩展拼写检查
BB-Code 文本格式
