# Hunyuan3D
## 安装pytorch
在 Anaconda Prompt 中，使用cd命令切换到目标路径D:\Hunyuan3D   
```
cd D:\Hunyuan3D
```
要创建一个名为Hunyuan3D且安装 Python 3.8 的环境
```
conda create --name Hunyuan3D --prefix D:\Hunyuan3D python=3.8
```
创建完成后，可以使用conda activate命令激活新环境
```
conda activate Hunyuan3D
```
## 检查 CUDA 版本
- pytorch官网: <https://pytorch.org/get-started/locally/>
