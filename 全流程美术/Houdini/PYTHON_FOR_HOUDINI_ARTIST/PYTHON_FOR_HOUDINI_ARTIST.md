# REBELWAY - PYTHON FOR HOUDINI ARTIST
[HOUDINI FOR 3D ARTISTS](https://www.rebelway.net/houdini-for-3d-artists)      

### 课程核心目标
学会用HDAs/程序化工具做环境资产（如货架、步道），掌握Solaris的USD导出+Karma渲染，结合Copernicus（程序化框架）、Megascans纹理实现高效材质控制。
这是一门基于Houdini的环境创建课程，核心是用HDAs（ houdini数字资产）和程序化工具搭建、修改3D场景，适配Solaris渲染和Karma引擎，课程分8周逐步推进：

### 每周内容总结
1. **第1周**：项目概述+技术思路，完成项目初始化，搭建工作流基础。
2. **第2周**：用程序化建模做平台主体，制作可复用的货架HDA；为Solaris渲染做场景准备，用HDA+Megascans生成材质。
3. **第3周**：步道系统搭建，用曲线生成基础步道，建立自动生成楼梯的工作流。
4. **第4周**：完善步道HDA，学习“几何体附着到原始元素”的实例化方式，替代传统“点复制”，实现步道元素的随机化。
5. **第5周**：基于距离/角度生成楼梯，扩展HDA以生成桥梁、拱门，完成步道系统的USD导出准备。
6. **第6周**：制作其他环境资产，利用已有工作流快速搭建高细节、可渲染的几何体。
7. **第7周**：用程序化纹理工具，学习纹理分层、操作组合，导入模型/属性并配置材质参数以强化细节。
8. **第8周**：导入动画角色+相机到Solaris，完成环境的材质分配、分镜头灯光与渲染设置。
